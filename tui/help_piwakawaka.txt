PIWAKAWAKA Viewer Help

MOUSE INTERACTIONS:
• LMB: Window level adjustment
• Mouse wheel: Scroll slices
• MMB: Translate scene
• RMB: Scale scene

KEYBOARD SHORTCUTS:
• h - Show this help window
• . - Add point/spline point
• u - Remove last point
• r - Reset view
• p - Save screenshot
• w - Reset window level

ARROW KEY NAVIGATION:
• Left Arrow - Previous time step
• Right Arrow - Next time step
• Up Arrow - Forward slice
• Down Arrow - Backward slice

SPLINE WIDGET INTERACTIONS:
Once created (3 points in spline mode):
• LMB: Move a spline handle
• Shift+RMB: Add a new spline handle
• CTRL+RMB: Remove a spline handle
• MMB: Translate spline
• RMB: Scale spline

MARKUP MODES:
• Point Mode: Add individual points
• Spline Mode: Create spline curve
• Closed Spline: Closed loop in the slice plane vs open polyline (not related to time).

SPLINE TIME INTERPOLATION (4D cine):
• Build manual splines on the slice you need at two or more time steps (orange curve = manual keyframe).
• Click "Interpolate splines" to fill every other time step; new curves are automatic (cyan) and are replaced on the next interpolate.
• Adjusting an automatic spline (move handle, etc.) promotes it to manual (orange) so it is never overwritten by interpolation.
• Only manual keyframes are used to compute the in-between splines (linear interpolation in time per curve sample).
• "Periodic temporal interpolation": time wraps over one cine cycle (last frame connects to first) when interpolating; leave off for acyclic time (e.g. breath-hold).

VIEW CONTROLS:
• Use slice orientation to switch between views

CUSTOMISED BUTTONS:
• Set custom actions by subclassing TUI - see online documentation

ANIMATION CONTROLS:
• Play / stop ANIMATION
• Use speed slider to change animation speed
